# 微信小游戏提审后台清单

按顺序完成以下步骤，可与 `COPY.md`、`PRIVACY.md` 对照使用。

## 一、素材上传

| 素材 | 路径 | 规格 | 上传位置 |
|------|------|------|----------|
| 小游戏图标 | `images/icon-512.png` | 512×512 PNG | 微信公众平台 → 设置 → 基本设置 → 头像/图标 |
| 分享图 | `images/share-500x400.png` | 500×400（5:4）PNG | 代码内 `G.SHARE.imageUrl` 已指向该文件；部分后台版本可在「分享设置」单独上传 |
| 宣传截图 1 | `docs/submission/screenshots/01-home.png` | 竖屏 9:16 | 版本管理 → 提交审核 → 宣传图 |
| 宣传截图 2 | `docs/submission/screenshots/02-gameplay.png` | 竖屏 9:16 | 同上 |
| 宣传截图 3 | `docs/submission/screenshots/03-clear.png` | 竖屏 9:16 | 同上 |

> **建议**：提审前用微信开发者工具真机预览，对主页、对局、结算页各截一张替换上述占位图，通过率更高。

## 二、基本信息

- [ ] **类目**：休闲 → 益智（或平台推荐的「休闲益智」）
- [ ] **简短介绍**：见 `COPY.md`（≤30 字）
- [ ] **详细描述**：见 `COPY.md`
- [ ] **版本号**：0.1.0
- [ ] **更新说明**：见 `COPY.md`

## 三、隐私与合规

- [ ] 设置 → 用户隐私保护指引：声明**本地存储**用途（进度、设置），参考 `PRIVACY.md`
- [ ] 确认未勾选未使用的敏感接口（定位、相册、手机号等）
- [ ] 当前版本**未启用**开放数据域排行榜（`leaderboard: false`），无需上传开放数据域工程
- [ ] 当前版本**未启用**音效（`sound: false`），无需上传 `audio/` 资源

## 四、代码与包体

- [ ] 微信开发者工具 → 上传代码，填写版本号 0.1.0
- [ ] 预览包体大小（主包建议 < 4MB；`levelpack.js` 较大，关注分包若后续需要）
- [ ] 运行 `node scripts/test-levels.js`，确认 47/47 关卡可解

## 五、提审前自测（真机）

- [ ] 新用户：教学 → 普通关 → 进阶机制教学 → 正式关完整流程
- [ ] 星星关：未收集星星无法通关
- [ ] 设置 → 清除进度：数据归零
- [ ] 小屏机：主页、成就页可滚动
- [ ] 结算页「分享」→ 右上角 ··· → 分享卡片含标题与配图
- [ ] 无崩溃、无明显卡顿

## 六、提交审核

- [ ] 版本管理 → 开发版本 → 选为体验版 → 内部体验确认
- [ ] 提交审核，填写测试账号说明：**无需账号，直接游玩**
- [ ] 审核备注（可选）：单机益智，数据仅存本地，无内购无广告

## 七、审核通过后

- [ ] 全量发布
- [ ] 监控用户反馈与崩溃日志（微信后台 → 运维中心）
