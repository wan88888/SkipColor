## 分享深链格式

| 模式 | query 示例 |
|------|------------|
| 普通关 | `mode=normal&level=5` |
| 每日挑战 | `mode=daily&seed=20250623` |
| 进阶机制 | `mode=advanced&mech=portal&level=3` |

好友从分享卡片进入后，会自动跳转到对应关卡（需满足教学解锁条件）。
