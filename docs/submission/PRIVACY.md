# 跳跃填色 · 隐私保护指引

**生效日期**：2026-06-23  
**适用产品**：微信小游戏「跳跃填色」（SkipColor）  
**版本**：0.1.0

## 1. 概述

「跳跃填色」是一款单机向休闲益智小游戏。我们重视您的隐私，本指引说明本游戏如何处理（及不处理）您的信息。

## 2. 我们收集的信息

### 2.1 当前版本（0.1.0）不主动收集个人信息

本版本**不注册账号、不收集姓名/手机号/身份证号**，也**不上传游戏进度至自有服务器**。

### 2.2 本地存储（设备端）

为保存游戏进度与偏好，游戏会在您的设备上使用微信本地存储（`wx.setStorageSync`），可能包括：

| 数据项 | 用途 |
|--------|------|
| 关卡进度、星星、成就 | 续玩与统计 |
| 音效/振动/粒子开关 | 个性化设置 |
| 主题选择 | 界面偏好 |
| 每日挑战日期与成绩 | 每日模式 |

上述数据**仅保存在您的设备本地**，卸载或清除小游戏数据后将丢失。

### 2.3 微信平台能力（由微信提供）

当您使用微信分享、振动、剪贴板（关卡编辑器复制关卡码）等功能时，由**微信客户端**按[《微信隐私保护指引》](https://weixin.qq.com/cgi-bin/readtemplate?t=weixin_agreement&s=privacy)处理相关数据。本游戏不额外获取您的微信好友列表或聊天记录。

### 2.4 当前版本未启用的能力

以下功能在 0.1.0 中**默认关闭**，启用前将另行更新本指引：

- **开放数据域好友排行榜**（`G.FEATURES.leaderboard = false`）：若开启，将向微信开放数据域写入匿名游戏分数，用于好友排行展示。
- **云数据库同步**：已移除，当前版本不使用。

## 3. 信息的使用目的

本地存储的数据仅用于：

- 恢复游戏进度与设置
- 展示成就、每日挑战与无尽模式记录
- 生成分享文案中的通关统计（在您的设备上计算，非上传）

## 4. 信息的共享与披露

我们不会向第三方出售或出租您的个人信息。除法律法规要求或微信平台的必要接口外，本游戏不向外部服务器传输用户数据。

## 5. 您的权利

您可以通过以下方式管理数据：

- 游戏内「设置 → 清除进度」重置存档
- 微信「我 → 设置 → 通用 → 存储空间 → 小程序/小游戏」清除本游戏缓存

## 6. 未成年人保护

本游戏为全年龄向益智内容，不含社交聊天、付费内购或不适宜内容。建议未成年人在监护人指导下使用微信及本游戏。

## 7. 联系我们

如有隐私相关问题，请通过微信公众平台后台配置的客服渠道或项目维护者邮箱联系（提审时在后台填写）。

## 8. 指引更新

功能变更（如启用排行榜、账号体系、统计分析 SDK）时，我们将更新本文件及微信后台「用户隐私保护指引」配置，并在必要时征得您的同意。

---

**提审说明**：请在 [微信公众平台](https://mp.weixin.qq.com/) → 设置 → 基本设置 → 用户隐私保护指引 中，勾选「本地存储」相关说明，并粘贴或链接本文件要点。当前版本无需配置「收集用户信息」类接口权限。
